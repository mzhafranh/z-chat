## Javascript Challenge: Making a Chatting App
Tools used:
1. Javascript
2. Node.js
3. MongoDB
4. Tailwind CSS
5. Express
6. NextJS
7. SocketIO
8. React

## Features
1. Real-Time Chatting
2. Real-Time Notification
3. Markdown and Code Formatting
4. Chat Editing
5. Resend messages when disconnected to server 
6. Responsive UI
